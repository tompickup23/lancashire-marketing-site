This repo turns the Lancashire civic-pride and tourism research into a GitHub Pages-ready website.

- `content/lancashire_borough_strategy.md`
- `content/lancashire_reserve_pool.md`
- `content/lancashire_brand_platform_and_talent_strategy.md`
- `content/lancashire_borough_influence_and_business_map.md`
- `content/lancashire_civic_pride_footage_research.md`
- `content/lancashire_county_council_division_footage_breakdown.md`

- `content/lancashire_public_talent_map_burnley_padiham.md`

- `content/burnley-social-following-research-2026-03-17.md`

## Local run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

The repo includes a GitHub Pages workflow in `.github/workflows/deploy.yml`.

You can also publish manually with:

```bash
npm run deploy
```
